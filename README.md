# Extensión Simbólica del Índice de Moran Global y Local para Datos Espaciales de Área

## Descripción

Este repositorio contiene los datos, scripts y resultados asociados al trabajo de grado desarrollado en la Pontificia Universidad Javeriana.

## Estructura del repositorio

```text
Tesis-Moran-Simbolico/
│
├── data/
├── scripts/
├── results/
├── manuscript/
└── docs/
```

## Reproducibilidad

1. Ejecutar los scripts de preprocesamiento.
2. Construir las matrices de vecindad.
3. Calcular los índices de Moran clásicos.
4. Calcular los índices simbólicos propuestos.
5. Ejecutar las simulaciones.
6. Generar tablas y figuras.
